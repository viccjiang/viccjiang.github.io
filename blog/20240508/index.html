三個學習重點，讓輸出發揮綜效

By Jiang 08 May 2024

這是一篇關於直播班的學習心態分享。

這次比較特別的是開始使用 ChatGPT、notion AI 來輔助學習，讓學習輸出時更加有複利效應。

有時候覺得自己怎麼沒有進步的時候，回頭看看以前的寫得 code 會覺得自己進步很多，很多以前看不懂得 code 現在可以很快就掌握起來。

最主要三個學習重點

  1. 課前預習課程 → 跟上老師節奏 → 課後複習課程

  2. 每週任務的刻意練習 ←→ 小步測試 ←→ 反覆驗證

  3. 充分運用課程提供的額外資源

    • 作業批改 — 盡可能修改助教給的建議
    • 每日的助教問答 — 可以在 DC 群盡情向助教們發問
    • 小組討論 — 跟組員一起動腦討論，會意外獲得更多
    • 每日任務 — 每天的小任務提升信心
學習心態的轉變
  1. 現在看到程式報錯不會緊張
  2. 遇到困難知道查資料的方法 ex. 如何下這個關鍵字、stackoverflow、ChatGPT、notion AI
  3. 進步，是隨著時間一步一步累積，急不得，挫折感是必然，成長是必須
  4. 學習看官方文件，隨時查文件
  5. 遇到較為困難的地方可以先做筆記
Programming Is 10% Writing Code And 90% Understanding Why It’s Not Working
  1. 看懂這行 code 的 報錯提示
  2. 知道每一行的報錯大概在幹嘛
  3. 報錯時，看懂行數然後再做求證
  4. 最常犯的錯是打錯字，強烈建議安裝拼字套件
  5. 次要犯的錯是 api 寫錯，導致接不到 api
  6. 很推薦新手先看 JS Error 參考資料，可以省去很多懷疑自己的時候

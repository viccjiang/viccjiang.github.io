從想到做培養原子習慣

By Jiang 01 February 2021

其實這裡都是寫給自己的心靈雞湯，隨時補充能量用，因為在朝前端努力的同時，正向思考很重要！

每天都進步 1%，一年後，你會進步 37倍； 每天都退步 1%，一年後，你會弱化到趨近於 0！ 一點小改變、一個好習慣，將會產生複利效應， 如滾雪球般，為你帶來豐碩的人生成果！

不知不覺已經一月底了，面對自己的學習進度顯得有點慚愧，本來預計這個月要把JS學徒學習一遍再加上小杰歷險還有學習Figma，但JS看到後面幾章節時根本鴨子聽雷，結果又回到前面重新學習了！又是一次砍掉重練的經驗！沒關係，只能先安慰自己，在新手村時就是要不斷的練習！ 另外，洧杰老師也在上週開始釋出JS直播班課前預習的章節，開始緊鑼密鼓的跟進，課前作業也持續交出去，目前跟著JS直播班課前釋出的課程，觀念有比較清楚，但目前都是片段的觀念，尚未整合起來！ 再來就是Figma的部分，目前斷斷續續的學習，主要是先聽UIUX觀念的部分，實作還沒開始著手，雖然自己有設計的底子，但是使用新的工具也是需要一點時間摸索跟習慣！

學習狀況的小結論：

最近學習時很容易分心，尤其是天氣太冷的時候很容易怠惰，而且專注力目前還不夠！

每天訂定目標，會讓學習比較快速進入狀況，要盯進度才會往前！

在家學習分心的主因之一是"貓咪"，而且是三隻貓咪，看到奴才在努力學習時，會不時的來蹭來蹭去，跳到電腦鍵盤上，實在太可愛…

年關將近，公司變化的波動比較大，一切等年終入袋再說！

感覺在過年前學習的心情很浮躁！

每個小結做一個小筆記

每週至少要做一個小作業

每週至少要安排一天休息

前端學習的短中長期計畫：

目前還是兼職在學習，所以學習時間為碎片化，依照洧杰老師的建議，將自己訂定短中長期目標，以「週」為單位來進行，較不會給自己太大壓力，每週盡可能的趕上進度就可以了！

短期目標 1~3個月

網頁切版：將 UI 設計稿轉化成 HTML、CSS 網頁版型，並會手寫 RWD 響應式網頁設計，或者使用 Bootstrap 4 框架達成

中長期目標 3~6個月 JavaScript：了解各 JS 面向，並掌握 SPA 三大框架其一 (Vue、React、Angular)

又想到一些小故事，突然想起N年前，大概是25、26歲吧，處於一個超級迷惘的時期，大學研究所畢業於休閒運動管理學系，覺得自己是個廢人，什麼都學但什麼都不會，而我是從一個什麼設計技能都不會的平庸人，經過5–6年的努力之後，轉職為平面設計師，過程是什麼樣子的呢？老實說我花了兩到三倍力氣在學習，因為這一路實在太顛簸崎嶇了，一開始目標很模糊的我挑了設計補習班的課程，當時好像選了跟室內設計跟遊戲設計之類的課程(完全跳過網頁設計、前端設計，是完全不考慮的那種)，現在回想起來覺得自己好傻好天真，覺得好燒錢跟時間呀，如果當時知道自己要的是什麼，一切可能就會不一樣吧，經過補習班的洗禮之後，覺得自己好像還很不夠資格當一個設計師(當初還去了室內設計公司做了一個月，結果發現自己完全不喜歡，而且主管質疑設計能力，做的超痛苦，反而又激勵了我)，索性就想再拿一個大學設計學位來證明自己的設計能力，然後就去報名了北科工業設計系(當時已經28歲了，嘆)，於是開始了兩年的宜蘭台北通勤工作與念書的日子，這些年的日子都是我成為平面設計師的養分，若沒有這些年的學習磨練，就不會有現在的我吧，至少有一個堪稱有謀生能力的技能，前前後後花了五六年在自我轉型，大概是想超越什麼吧，對，就是當初那些當頭棒喝的那些話，所以我一定會超越自己超越你！ 在還年輕時，挑戰自我極限，不設限！就是要做自己喜歡的、擅長的，這樣才能細水長流！當然，認識我的人，都知道我一直保持著自己對製作皮革這項興趣的熱忱，甚至已經可以製作精美的產品販售給喜歡皮件的人！學會一門技術不難，難的是保持熱忱，學習前端就像我對皮件的熱忱一樣，我會繼續保持下去，今年已經過了12分之1了，真的過得太快，更要珍惜「時間」這個寶貴的資產！

善用 ORID 焦點討論法，讓思考流動

By Jiang 02 February 2021

也可當作提問框架或 寫作的技巧

為了方便學習記憶，最近開始在寫一些學習筆記或讀書心得，結果發現自己寫得很零散，沒有一個架構或是段落，所以上網估狗了一些做筆記的方法，發現了一個不錯又好上手的方式，結果發現這筆記的方式也是提問的技巧：

ORID 焦點討論法

也可以當作 提問框架 / 寫作技巧，以我個人學習或閱讀書籍為例子，每天的學習筆記或讀書心得，可以根據這四個大方向來進行：

事實 -> 感受 -> 想法 -> 決定

Objective 事實

學習： 關於今天的課程，你記得什麼？ 完成了什麼？

閱讀： 這本書裡你看到了什麼？ 記得哪些部分？

Reflective 感受

學習： 你要如何形容今天學習的情緒 今天的學習高峰是什麼? 今天的學習低點是什麼?

閱讀： 這本書有哪些部份讓你最有印象的？ 讀了這本書的感受是什麼？

Interpretive 想法 學習： 我們今天學到了什麼？ 今天學到一個重要的領悟是什麼？

閱讀： 書中印象深刻的原因？ 為什麼會有這樣的感受？ 你想到了什麼經驗？ 重要的意義是什麼？

Decisional 決定 學習： 我們會如何用一句話形容今天的學習？ 有哪些學習需要明天繼續努力?

閱讀： 有什麼值得改變或應用的地方嗎？ 接下來你的行動/計劃會是什麼？

真正有效的學習是「輸入」與「輸出」並行

也就是將一連串 Input 後的內容，內化歸納後 Output 成果！所以很多大神的作法都是建議將學習過程

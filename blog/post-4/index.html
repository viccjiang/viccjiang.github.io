後防疫時代，進修學習不間斷

By Jiang 16 June 2020

目前學會套用 Hugo 現有主題， 完成一個個人網站部落格並且更進階的運用在台大學到的前端相關知識， 在現有的 Hugo 主題基礎上，修改出自己的個人網站， 在學習的其間也學會Markdown 語法，對於做筆記與寫部落格很實用！

前陣子2月3月疫情還很嚴重的時候，沒上班時幾乎都宅在家裡投資自己，嘗試做一些在家的靜態活動，像是看書、看線上課程、做手作販賣、商品攝影等等等，還有很多在家裡可以做的事情！五年前因為工作的關係，因而埋下了程式語言的種子，開始對於程式語言超級感興趣，很想自己寫個什麼東西出來，尤其是架設官網或建置部落格，於是開啟了程式語言學習之旅。

老實說，以前買過很多關於程式設計的線上課程，但發現自己對於線上教學都很容易分心也沒什麼耐心，也沒有做出什麼太大的突破，但前陣子買了Github 免費架站術！輕鬆打造個人品牌這門課，發現整部課程真的很適合我，每個單元都很迷你且重點超濃縮，很符合各種需求，老師也都會回答各種問題，真的很棒呢！這門課我只花了一個禮拜就看完了，而且是用下班後的時間，課程吸收程度很好，最後也把部落格架起來了，算是達到一點點程式語言學習的小里程碑！

現在六月了，也終於等到疫情趨緩，決定去上實體課程，希望利用實體課程的壓力來督促自己要更進步，於是在上個月報名台灣大學資訊系統訓練班的課程，其實老在三年前已經上過一次，但當時都是下班後再去上課，上課內容吸收得很不好，後期就呈現半放棄狀態，很多css都寫不出來，版也切不出來，所以這次學聰明，報名假日班，睡飽再去上課，上課內容吸收效率極好，再加上前陣子買了 Hahow 的 線上課程 GithubPages 課程做搭配，台大跟Hahow目前都已經完課了，也分別把台大的作品以及hahow的作品部署上線，這次花了兩個月的時間在學習前端語言，覺得收穫滿滿滿，尤其是發現在實體課程班上有國中生跟高齡者之後，真的覺得自己要更上進了，這周也要開始上其他的課程，滿心期待！

實用語法筆記 :

  • Visual Studio Code 編輯器 - 可直接在 VS code 進行編輯

  • cmd - window 命令提示字元 ( 直接在資料夾路徑key入 )

  • terminal - mac 終端機 terminal ( 資料夾右鍵服務選項也可開啟 )

  • 輸入hugo version 可查看 hugo 版本

  • 輸入hugo server 啟動 hugo

  • 輸入hugo 建立html的資料夾

  • public 是建立html的資料夾後的生成的資料夾，資料上傳到 GitHub Page 進行佈署

六角學院網頁切版直播班心得 - 2020年秋季班

By Jiang 26 December 2020

先來個前言吧！

時間真的過得好快，八週的切版直播班竟然就這樣結束喏（兩個月真的太短了啦），而且，再過幾天就就正式邁入34歲了(歲月是把殺豬刀)，轉職之路也正式展開了，前幾週看了S姐大神來六角的影片真的滿焦慮的，提到28歲到38歲是公司最愛的年紀，是用來轉職最好的時間點，而我已經要34歲，似乎沒多少時間準備了，目前預計明年是最後的學習衝刺與機會，期待自己一年後能成為更好的樣子！

回顧過去五年，在2015年時，我進入了一個多媒體工作室，主要是招攬業者的大型入口網，負責職務是網站美編，換換圖改改圖上上文章PO貼文，大概就是這樣的乏味的工作，直到有一天資訊部主任叫我改個文字顏色，才發現原來更改語法是這麼有趣的一件事情，畫面也跟著有變化，再加上當時的老闆因為被資訊部的人嗆外行不懂，結果老闆整個被激怒到，開始自學程式語言，後來那位老闆也成為一名前端工程師，這才發現原來懂程式語言的人，還滿狂的，我也是滿震撼，於是開始對於「前端工程師」這個職位很有興趣，就這樣對這個職位嚮往到現在~

當初為什麼報名這堂課？

介紹一下背景好了，我目前是平面設計師，約莫在2016年~2017年處於一個對未來迷惘的階段，一直還在摸索自己的潛能，開始搜尋工程師、前端、程式設計這幾個關鍵字，結果一直被六角學院廣告打到，基於很好奇這個領域，於是乎我終於在2016年年底時買了第一套六角學院的課程(HTML/CSS/jQuery/RWD)，但是… 超沒動力的，到了2019年始終沒看完，直到2020年初疫情爆發，才開始意識到「待在家裡，投資自己」的重要性，大概2020年1月時就開始把一些基礎的課程拿出來看 HTML/CSS/jQuery/Bootstrap … 但因為第一屆切版班開班時還沒把課程看完，也害怕自己跟不上進度，所以就放棄報名了，在放棄第一屆切版班的這幾個月裡，有去參加台大資訊訓練班的實體課把一些基礎打起來！另外，因為自覺自己是容易發懶的人，需要督促也需要老師有系統化的教學與鞭策，直播班對我來說會是最好的選擇，也終於盼到第二屆開班通知了 ％％％％％

八週學習過程全記錄

雖然自己不是最優秀的學員，但期望自己的學習過程分享也能帶給大家一點參考價值！

第一週 Padding / Margin / 盒模型概念

終於等到 網頁切版直播班 開課了，在還沒開課之前，有先把課前釋出的課程看完並且交作業，也有到坊間進修一段時間，有到台大上課，當然也有買了好幾堂六角學院的課程(今年才發現是2016年買的，都沒認真看完)，也有買了Hahow 的前端課程，直到今年初疫情爆發，只好待在家裡投資自己，把一些重點課程看完，大致上在2020前半年打下了一些基礎，所以在直播課程過程中，比較沒有太大障礙，

直播課程的好處在於，當下有問題可以提問，而且每位同學都會是你的老師，不管是在zoom或是slack上都可以很快得到回應！

從2016年跟著老師到現在，發現六角學院的社群力量真的很大，常常看到很多學長姐在社團發問，同學老師都會很熱心的回答，尤其是不排斥新手發問，這點對於新手真的很重要，這是我喜歡在六角學院線上學習的理由！

第二週 Flex 排版

這週開始有點難度了，但如果有認真看過課前釋出的flex影片，就可以很快跟上進度，這週的小組任務很可愛，要玩兩個flex的遊戲 Flex pirate 還有 Flex 青蛙，玩到後面真的有點小困難，需要思考一下才有辦法過關！上週開始有練習英打速度，一開始真的很卡，但是連續練習三天就有點順起來，但是又遇到假日就又怠惰了QQ，預計本週每天都要練阿~

第三週 RWD 響應式網站 先看權重再看先後

這週二加碼穎旻助教的直播，又打通了一些平常會犯錯的概念，收穫很多，也學會了一些前兩周Adobe XD觀看圖稿的技巧，看老師在講解作業時都感覺很沒問題，先看權重再看先後，但自己在寫作業的時候就整個GG了，雖然都有一直在複習之前影片，但自己寫RWD時還是一直在跑版，code也超級無敵亂七八糟，結果發現自己寫錯斷點992寫到768去，一整個大亂套！邊上班邊直播切版班真心滿累的，自從開課以來，就沒有一天睡飽過，收穫很滿！總之，RWD大魔王還沒打敗，各位一起加油了！

第四週 多頁式管理 / 自動化流程 / SCSS

這週的直播學習開始導入Gulp自動化流程和SCSS系統，看似很容易，但在一開始安裝時就頻頻卡關，好險有老師與助教寫的手冊，暫時解決卡關的問題，mac很多時候都要加sudo，又學會一招了，週二助教開的直播加碼，老實說開始有點聽不太懂，有點消化不良，下班回家又重新看了一遍和聽一遍，觀念還是覺得很模糊，本來第四週要開始用Gulp和SCSS寫看看，結果不知從何下手，而且時間上的不允許，沒辦法再打掉重練了，大概是因為前面基礎都還沒打穩健吧，決定要重看影片再來試試看，這週就先把第三第四週寫個兩頁RWD了，Gulp和SCSS持續研究中～最後，我們小組有完成四週的小組任務，也可以獲得勇者徽章囉

已經收到實體勇者徽章啦 XD 非常榮幸 rrrrr

第五週 Form 表單 / SCSS / Bootstrap

這週手刻了一個後台表單設計，真的好難啊，頭快要炸了，目前還不太會什麼 OOCSS BEM 之類的，Bootstrap元件也還不會使用，所以很單純地用自己這幾週所學的會的語法拼湊出來，上周的 gulp & scss也都還沒搞清楚，這周末要繼續拚之前還沒搞懂的東西（ 更新 11/21 順利地 gulp build -> gulp deploy)，但上傳過程大概搞了四個小時，中間一直有問題，參考了Ray助教的 Gulp教學，最後成功了!這大概是這週最有成就感的事情吧，對於gh-pages 又多了一層了解！最後發現一直出錯是因為.git檔案沒刪掉@@ 還有出錯的問題點一直圍繞在sudo，sudo你真的好煩呀！

第六週 Bootstrap / Gulp 部署

上週作業嘗試用了 Gulp 部署到 Github Pages 發現相當方便且快速，相較於Github Desktop，Gulp 真的快速許多，雖然要記的指令很多，順序也要記一下，雖然上傳失敗很多次，但最後成功了真的很令人振奮，為此做了個自己在部署時的筆記 Gulp gh-pages 部署筆記。這週的作業也是用 Gulp 來開發，除了可以用 layout.ejs 之外，scss 也不用額外用 live sass compiler 編譯，真的很省事，所以這週很認真地用 Bootstrap 的格線系統與元件來做作業，發現很多地方都跟 flex 脫離不了關係，學 Bootstrap 之前真的要先把 flex 的基礎打好，不然直接來看 Bootstrap 應該會一頭霧水，說到這裡，猛然發現原來已經第六週要邁向第七週啦，直播班真的過得好快好充實，自從直播班開課以來，除了固定的週五直播課以外，每天晚上就是安排自己與程式語言搏鬥的時間，看了一下自己的行事曆，每晚排滿滿的學習進度，這兩個月來沒有間斷過，這也是這兩個月給自己的一個里程碑，既然要花錢，就要花得有價值，要繼續努力完成這門課程，也要做出幾個像樣的作品。

第七週 AOS / animation / 視差滾動

跟著洧杰老師的腳步終於來到第七週了，這週課程也相對歡樂，相較於 RWD Bootstrap Gulp 是簡單多的 CSS 動畫效果，在 AOS 視差滾動效果裡 background-attachment: fixed 這個語法非常重要，主要是固定背景的方式，來達到視覺滾動的效果，這次的作業練習也有使用到這個語法，另外，第一周到第七週作業完成度也達到70%了，目前對於自己的行動力感到相當滿意，但是對於未來轉職之路還是很迷惘，今天也在slack上詢問了穎旻助教目前前端職缺的看法，覺得稍微有解惑，也相信自己一步一腳印，一定可以達成的！最後一週，準備衝刺囉，六角學院提供了好多獎品啊，一定要拿到！這就六角學院是跟一般仿間補習班不同的差別，六角學院就是這麼有溫度呀！老師最後公布了所有獎項，天哪也太佛了，完成率90%就可以參加下一屆的直播課程，大家加油呀，一定要得到，非常超值!

第八週 最後一週 衝一波

最後一週啦，作業真的要衝刺了，上週看到老師在 slack 發的心得文，感到相當激勵，因為在這裡你會發現很多人跟你一樣都在為未來做準備，你不是孤軍奮戰，小組也有人在跟你一起努力，互相鼓勵是很必須的，走到這步時，也發現自己的刻意練習也練習了兩個月，每天上班第一件事情就是開slack做每日任務10~15分鐘，晚上下班一定會複習前一週的影片，每天至少兩小時不斷的複習與練習，會發現自己進步了很多，我們大家都走到第八週這一步了，接下來才是真正的開始 ~ 最後一週版型實在太可愛，一定要做完!!!( 目前還在努力中… 希望可以完成2-3頁 )

最大的收獲是？

中文授權版型、切版技巧、Bootstrap、Gulp多頁設計、表單設計…… 還有無限的 code review 與 老師助教的回饋

最重要的是這屆老師終極加碼辦了線下實體活動呀 ( 排除萬難手刀報名 )

可以見到老師還有組員的廬山真面目，還可以讓洧杰老師現場親自code review ( 這是最大最大的收穫吧！結果我的code讓老師釘到爆 ) ,在這場活動裡認識一些各路業界大大，有餐飲廚師、後端工程師、文案設計師、PM… 結束後也互加 line ，讓我覺得學習路上不是孤單的！

最喜歡直播班的哪些活動？

Slack討論串、每週主線任務、每週直播課、每週助教加碼課、每日任務、每週小組討論，BOSS遊戲系統超酷(記得是星期三星期五會出來嚇一下大家)

其實很喜歡助教的每週二更詳細解析直播課程，因為每次週五上課時都是我一週最累的時候( 上班族的下班後真的只想耍廢，常直播上課上到睡著 )，好險每週二還有助教加碼課可以再更進階的惡補一下！

老師與助教改作業都會解析這週主線任務版型的建議作法，我覺得這點很棒，會給學生一個雛形，讓學生自己再去建構與組織

這次分組是分到設計魔劍組，應該是一群設計師設計人的組，但大家好像上班都很忙碌比較少出現，每次討論的大概也是固定2-3人(我、小組長、另一個組員)而已，但每週小組都還是有討論並且互相激勵，也會分享自己實作的部分！最後感謝小組長每周都會召集開討論！

如果時光能倒流，會希望自己再次注意哪些細節？

  1. 把握八週時間，瘋狂提交作業!!!!!
  2. slack 要每天看大家提的問題，並且認真記錄(slack過一段時間會洗掉之前紀錄)
  3. 多看別人的原始碼做學習

覺得還可以改善的細節？

因為自己目前是上班族，時間真的很有限，雖然作業都有如期繳交，但是總覺得自己交得很倉促，code 寫的很亂，雖然老師都說這是必經過程，會從爛code進化到好code！最後我發現每週給自己一定要交作業的時程壓力，才真的會進步，我的策略是「與其一週卡住不交，不如每週先交再說」，因為老師跟助教會給很多很多的回饋，再針對這些回饋去做優化就可以了！直播上課時的輸入與每週主線任務的輸出，就可以知道自己卡住的問題點在哪，以及該優化的地方！

身為學長姐，分享些想入坑的新同學一些勉勵的話：

  1. 直播互動性超高
  2. 作業爆炸多要有心理準備，但一定要交作業
  3. 除了直播課，助教還有無限加碼課，看不完rrrrr
  4. 六角學院 提供的完課獎品CP值超級高
  5. 保證教學品質，限量300位學生

八週切版直播課總結

文章的最後一定要個總結，所以我來總結一下我上完這堂課之後的感覺：

  1. 這個價位物超所值
  2. 老師助教群很優秀
  3. 課程設計很有脈絡
  4. 切版實戰經驗累積
  5. 六角同學也是老師

因切版直播班讓我的成長幅度很有感覺，已經立馬報名2021年的JS和VUE直播班了，開始期待未來一年後的自己！

走出舒適圈，突破自我與提升能力

八週課程很快地已經結束，但也才是真正的開始，這八週的養分讓我學會開始優化自己寫的code、設計流程、邏輯思考、解決問題等等等，洧杰老師這八週來也一直強調「你的問題也會是其他人的問題」，每當我卡關時，都會先到 slack 打關鍵字找答案，才發現原來很多人都有跟我一樣的問題，而且大家都會互相幫忙，正所謂「一個人走的快，一群人走的遠」，大概就是這個道理吧！我們就是這群人，一起走遠吧！
